# Academic Compass

A futuristic, glassmorphism-styled web application built with Vite, React, TypeScript, and Supabase.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create a `.env` file in the root directory:
   ```
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

3. Start the frontend:
   ```bash
   npm run dev
   ```

## Styling

This project uses Tailwind CSS and a custom color palette:
- Primary: `#132041`
- Secondary: `#193656`
- Accent: `#af1c1e`
- Dark Background: `#22232b`

The UI heavily relies on glassmorphism principles (translucency, blur, and subtle borders).
